Font Awesome Icons

How to use Font Awesome icons?

Step 1: Include the following Font Awesome CDN link in the <head> section of you HTML document.

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css">

Step 2: To incorporate icons into your web page use the syntax <i class="fa fa-*"></i> where * represent the class of a particular icon. Therefore, if an icon class is user then the icon HTML code would be <i class="fa fa-user"></i>, similarly if the icon class is address-book the resulting icon HTML code would be <i class="fa fa-address-book"></i>, and so on. See live example.
